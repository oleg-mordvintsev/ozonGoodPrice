# Поиск отличной цены на сайте Ozon
- Для работы не требует ничего, кроме браузера
- Запуск осуществляется через консоль
- Вкладка может быть свернута, т.к. при срабатывании правила издаётся гудок

## Редактирование товаров на мониторинге
- Перейдите на сайте в список товаров в нужной категории, где будет производиться поиск по цене 
- Сделайте упорядочивание по цене, от малой к большей
- Запишите адрес страницы в параметре Items.url
- Укажите цену сработки, дешевле которой необходимо вас уведомить Items.price
- Заполните параметр Items.words со списком слов через пробел, которые точно должны присутствовать в названии товара
- Добавьте заголовок срабатываемого правила, чтобы Вы сразу поняли, что сработало

## Пример заполнения
- Пример есть в самом файле
- Где в Items и добавляются нужные правила для срабатывания, в примере, все кроме первого правила закомментированы и сделаны для примера

## Как запустить?
- Поправить правила сработки для нужного товара
- Скопировать код в буфер обмена (копировать/copy/ctrl+c)
- Открыть браузер с главной страницей OZON или любой внутренней
- Открыть консоль браузера, для google chrome Ctrl+J
- Вставить код из буфера обмена и нажать Enter

## Как узнать что сработало?
- Скрипт работает по кругу один раз в 5-ть минут
- Если сработает какое-либо правило, то он остановится, подаст звуковой сигнал, будет мелькать красным фоном страницы Ozon и в консоле будет показано какое правило сработало со ссылкой для перехода к товару
